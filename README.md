# Nagarajan Mohan | Senior UAV Software Developer Portfolio

A premium, highly-interactive, and fully responsive personal portfolio website built for Nagarajan Mohan. This project showcases expertise in advanced drone technology, UAV software systems, and full-stack development.

The website was recently modernized to feature a sophisticated **Day/Night Theme**, a cohesive **Design System**, and a dynamic **Global Search Engine**, tailored to impress senior hiring managers and tech recruiters.

## ✨ Key Features

- 🌓 **Dynamic Theme System:** Seamlessly toggle between a premium Dark/Slate aesthetic and a clean, high-contrast Day Mode interface. UI elements intelligently adapt, ensuring maximum readability and visual impact.
- 🔍 **Global Search Engine:** Built-in instant-search functionality embedded in the navigation bar. Instantly locates keywords (like "Java", "Education", or "LUA") across all project and profile pages.
- 🚀 **Interactive Feedback:** High-end hover effects, scale animations ("Pop" effects), and CSS-driven smooth scrolling. 
- 📱 **Fully Responsive:** Grid and Flexbox layouts guarantee flawless display across mobile, tablet, and desktop devices.
- 📬 **Live Contact Integration:** Direct integration with Firebase SDK to securely collect form submissions directly into a Firestore database.
- 📁 **Detailed Project Pages:** Individual, categorized web pages for deep dives into Android, Web, Windows, LUA, and Hardware integrations.
- 🎓 **Professional Profile Grid:** A dedicated profile page cleanly highlighting Degrees (MCA/BCA), Certifications, and Years of Experience natively designed for recruiters.

## 🛠️ Technologies Used

### Frontend & Styling
- **HTML5 (Semantic)**: Modern structural tagging for accessibility and SEO.
- **CSS3 (Custom Variables):** Intelligent theming using `:root` tokens, glassmorphism (`backdrop-filter`), and CSS Grid/Flexbox architecture.
- **Vanilla JavaScript (ES6 Modules):** DOM manipulation, IntersectionObserver for scroll animations, `localStorage` caching for themes, and global search logic.

### Assets & Backend
- **Phosphor Icons:** Crisp, premium vector iconography automatically linked via CDN.
- **Google Fonts:** Utilizing 'Inter' for body readability and 'Outfit' for highly structured, modern headings.
- **Google Firebase (Firestore):** Serverless Backend-as-a-Service powering the secure, real-time Contact Form submission pipeline.

## 🚀 How to Run Externally

This site requires no compilation or build steps. 

1. **Clone or Download** the repository.
2. Ensure you have the localized `/Assets/` directory matching the HTML image references.
3. **Open `index.html`** in any modern web browser (Chrome, Firefox, Edge, Safari).
4. *(Optional)* To test Firebase Contact Form routing, ensure your local environment allows external API calls/CORS origin depending on your specific Firebase configuration.
